# 4_DE_Analysis

R Markdown scripts for differential expression analysis of CC pup RNA-seq data. Scripts take the count matrix from `3_STAR_Alignment/` as input and perform DESeq2/edgeR-based DE analysis with biological comparisons of interest.

## Analysis Scripts

| Script | Description |
|--------|-------------|
| `AllPups_Diet_DE_analysis.Rmd` | DE analysis across all pups comparing dietary treatment groups |
| `ControlPups_DE_Analysis.Rmd` | DE analysis restricted to control pups |
| `CCRIX_DE_analysis.Rmd` | DE analysis in CCRIX (reciprocal intercross) samples |
| `CCRIX_PhenoXGE.Rmd` | Phenotype × genotype interaction analysis in CCRIX samples |

## Workshop Materials

The `RNA_seq_workshop/` subdirectory contains teaching materials for RNA-seq DE analysis:

| Script | Description |
|--------|-------------|
| `DE_Analysis_mm.Rmd` | Introductory DE analysis walkthrough (mouse) |
| `enrichment_mm.Rmd` | Gene set enrichment analysis walkthrough (mouse) |

## Usage

Open any `.Rmd` file in RStudio and knit, or render from the command line:

```r
rmarkdown::render("AllPups_Diet_DE_analysis.Rmd")
```

## Notes

- GBRS-based allele-specific expression analysis has been moved to the [RNAseq_GBRS](https://github.com/Dpbudke/RNAseq_GBRS) repository
- Scripts assume the count matrix has been generated by `3_STAR_Alignment/4_reads2counts.sh`
